DLP_Proyecto.pdf – Documento principal del proyecto donde se desarrollan las políticas de seguridad DLP, clasificación de datos, acceso, monitoreo y concienciación.

Descripción del Proyecto

El proyecto se centra en:

Introducción a DLP: Qué es y por qué es importante proteger los datos sensibles.

Clasificación de Datos: Datos Públicos, Internos y Sensibles.

Acceso y Control: Aplicando el principio del menor privilegio.

Monitoreo y Auditoría: Herramientas y procesos para revisar el uso de la información.

Prevención de Filtraciones: Uso de cifrado, restricciones de dispositivos externos y controles de aplicaciones.

Educación y Concienciación: Capacitación del personal sobre riesgos y políticas de DLP.
